# Margarita App - Project Vision

## What This Is
Margarita App is a professional editorial design tool aimed at creators who seek a coherent visual identity. It allows users to extract color palettes from high-resolution images and generate custom calligraphy stickers.

This project is built upon the initial design flow generated by Figma Make (`Margaritaappdesignflow`), prioritizing **functionality over design** in its core development phase.

## Core Pillars (Functions First)
1. **Intelligent Color Palette (Staple):** Extract representative tones from images and interact with them via a draggable "staple" on the canvas.
2. **Pro Calligraphy Motor:** Upload custom fonts (.TTF, .OTF, .WOFF) and generate alpha-transparent PNG stickers in black or white.
3. **Editorial Composition:** Flexible canvas with standard social media formats (Story, Square, Feed).
4. **Cloud Persistence:** Secure login and project management via Supabase.

## Technical Architecture
- **Frontend:** React (TypeScript) + Vite.
- **Styling:** Tailwind CSS v4 (from base) + Vanilla CSS refinements.
- **Engines:** Browser Canvas API for color quantization and image-to-text rendering.
- **Backend:** Supabase (Auth & Storage).

## Success Criteria
- High-fidelity color extraction (5-6 colors).
- Smooth long-press + drag interaction for the palette.
- Crystal-clear PNG stickers with perfect transparency.
- Projects synced across devices.
