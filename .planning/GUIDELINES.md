# Margarita App - Design System Guidelines

## General guidelines
* Only use absolute positioning when necessary. Opt for responsive and well structured layouts that use flexbox and grid by default.
* Refactor code as you go to keep code clean.
* Keep file sizes small and put helper functions and components in their own files.

## Design system guidelines
* Use a base font-size of 14px.
* Date formats should always be in the format “Jun 10”.
* The bottom toolbar should only ever have a maximum of 4 items.
* Never use the floating action button with the bottom toolbar.
* Chips should always come in sets of 3 or more.
* Don't use a dropdown if there are 2 or fewer options.

## Button Variants
* **Primary Button:** Used for the main action. Bold, filled with primary brand color. One per section.
* **Secondary Button:** Alternative/supporting actions. Outlined, transparent background.
* **Tertiary Button:** Least important actions. Text-only, no border.
